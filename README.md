## 📖 Descrição
O Uber SafeStart é uma solução inovadora para aumentar a segurança no transporte por aplicativo, prevenindo riscos antes e durante as viagens. Desenvolvido como um aplicativo Android completo, ele promove o respeito mútuo entre motoristas e passageiros, combatendo situações de assédio e preconceito através de mecanismos inteligentes de prevenção e monitoramento.

## ✨ Recursos Principais

### 🛡️ SafeScore
Um sistema de pontuação que incentiva e recompensa práticas seguras:
- Pontos por completar checklists de segurança
- Bonificação por compartilhar rotas
- Recompensas por feedback construtivo
- Visualização clara da pontuação no perfil

### 👥 Pareamento Inteligente por Gênero
- Opção para mulheres e pessoas não-binárias preferirem motoristas do mesmo gênero
- Interface intuitiva para gerenciar preferências
- Implementação respeitosa que prioriza o conforto e segurança

### 🏆 Sistema de Conquistas
Gamificação para incentivar boas práticas de segurança:
- "Viajante Iniciante" - Complete sua primeira viagem
- "Sempre Alerta" - Use o checklist de segurança regularmente
- "Compartilhador Seguro" - Compartilhe sua rota com contatos
- "Especialista em Segurança" - Acumule pontos no SafeScore
- E dentre outras!

### 🎙️ Gravação de Áudio Consensual
- Gravação apenas com consentimento mútuo
- Armazenamento seguro por 30 dias
- Utilizado apenas em caso de denúncias

### ✅ Checklists de Segurança Interativos
Verificações específicas para cada perfil:

**Passageiro:**
- Verifique a identidade do motorista
- Confirme que está com cinto de segurança
- Compartilhe sua rota com contatos
- Verifique placa e modelo do veículo

**Motorista:**
- Verifique condições do veículo
- Confirme documentação em dia
- Aceite o Código de Respeito
- Opção de ativar gravação de áudio

### 🚨 Verificações de Segurança Durante a Viagem
- Alertas periódicos verificando se está tudo bem
- Opção de reportar problemas discretamente
- Confirmação de conforto e segurança ao longo do trajeto

### 📊 Feedback Pós-Viagem Aprimorado
- Perguntas específicas sobre a experiência de segurança
- Opção de reportar comportamentos inadequados
- Bonificação de SafeScore por feedback construtivo

## 📱 Telas Principais do Aplicativo

### 🚗 Fluxo do Passageiro:
1. Login/Cadastro com opção de gênero
2. Tela inicial com busca de destino
3. Visualização do motorista compatível
4. Checklist de segurança pré-viagem
5. Monitoramento durante a viagem
6. Feedback pós-viagem

### 🚘 Fluxo do Motorista:
1. Login/Cadastro específico
2. Modo motorista com status online/offline
3. Visualização de solicitação de corrida
4. Checklist de segurança do veículo
5. Interface de navegação durante a corrida
6. Feedback e avaliação do passageiro

## 🔒 Recursos de Segurança Adicionais
- Criptografia de dados na comunicação cliente-servidor
- Armazenamento seguro de dados sensíveis
- Sistema de denúncia com múltiplos níveis de verificação
- Visualização do local de embarque/destino com StreetView

## 🛠 Tecnologias Utilizadas
- **Frontend:** Java/Android (Android Studio)
- **Backend:** Node.js com Express
- **Banco de Dados:** SQLite (Room para persistência local)
- **Criptografia:** AES-256 para comunicação segura
- **UI/UX:** Material Design 3 com temas escuros
- **Mapas:** Google Maps API e StreetView Panorama

## 📊 Impacto e Benefícios
- Redução significativa de incidentes e casos de assédio
- Maior retenção de motoristas devido a um ambiente mais seguro
- Aumento da confiança dos usuários na plataforma
- Promoção de uma cultura de respeito e responsabilidade mútua
- Experiência de viagem mais tranquila e confiável

## 🚀 Como Instalar

### Método 1: Download Direto
1. Acesse as [Releases (UberSafeStartAPK)](https://github.com/2025-1-NCC3/Projeto8/releases/tag/UberSafeStartAPK) no GitHub
2. Baixe o arquivo APK mais recente
3. No Android, habilite "Instalar de fontes desconhecidas" nas configurações
4. Abra o APK baixado para instalar

### Método 2: Compilação do Código-Fonte
1. Clone o repositório
2. Abra o projeto no Android Studio
3. Compile e execute no emulador ou dispositivo físico
